Lab2 - Instrucțiuni de ramificare și ciclice în C

Lucrarea este dedicată utilizării structurilor de control (if, else, switch, for, while, do-while) pentru rezolvarea unor probleme variate. Sunt implementate aplicații pentru verificări numerice, caractere, triunghiuri, ecuații, precum și algoritmi bazati pe bucle, dezvoltând gândirea algoritmică.
