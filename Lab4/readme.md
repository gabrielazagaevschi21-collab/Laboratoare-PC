Lab4 - Tablouri unidimensionale și bidimensionale

Lucrarea conține un set amplu de probleme pentru prelucrarea tablourilor unidimensionale, implementate cu funcții iterative și recursive. Sunt realizate operații precum căutare, sortare, inserare, ștergere, rotații (ROL, ROR) și analiză statistică a elementelor, utilizând un meniu interactiv.
