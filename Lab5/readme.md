Lab5 - Utilizarea șirurilor de caractere

Lucrarea tratează prelucrarea șirurilor de caractere în limbajul C. Programul implementează funcții pentru numărarea consoanelor și cuvintelor, verificarea palindromului, inversarea șirurilor și a cuvintelor, precum și căutarea, ștergerea și înlocuirea caracterelor și cuvintelor, folosind un meniu interactiv.
